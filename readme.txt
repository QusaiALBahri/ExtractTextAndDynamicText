# Selenium Temperature Scraper

This Python script uses Selenium to automatically navigate to a webpage and extract temperature information.

## Description

The script sets up a Selenium WebDriver with customized options to maximize compatibility and performance. It navigates to a specific webpage designed to show temperature data and extracts this information cleanly from the page.

## Prerequisites

Before running this script, ensure you have Python installed on your system. You also need the Selenium WebDriver. This script is designed to work with Google Chrome, so make sure you have the Chrome browser installed along with the corresponding version of ChromeDriver.
